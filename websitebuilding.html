Setting Up Visual Studio Code

Install Visual Studio Code and open it. Then, add the Live Server extension to preview your work in real time. Create a new folder, open it in VS Code, and enable Auto Save by selecting File → Auto Save.

Understanding the CSS Box Model

Familiarize yourself with the Box Model by reading about it here (w3schools). Experiment with different sizing units such as px, vh, vw, and %. You can change background colors using background-color: or apply a background with background: url('https://via.placeholder.com/800x200') or url('assets/background.jpg') no-repeat center center; in CSS.

Let's Start Coding!

Create a folder named Assets and add three files: index.html, style.css, and script.js. Begin by pasting the boilerplate HTML into index.html and updating the <title> tag.

Publishing on GitHub Pages

1. Create a new repository on GitHub.

2. Name it your-username.github.io.

3. Set the repository to public.

4. Upload your project files.

5. Navigate to Settings → Pages, select Deploy from a branch, and ensure the correct branch is set (usually main).
